# Roadmap

See the main [ROADMAP.md](../ROADMAP.md) at the project root for the full roadmap.

This file links to the canonical roadmap and provides quick contribution guidance.

## How to Contribute to the Roadmap

1. Open a GitHub Issue with the `roadmap` label to propose a new item
2. Comment on an existing issue to claim a planned item
3. PRs that implement roadmap items are merged with priority

## Current Priority Items for Contributors

1. **OpenRouter provider** — adds a unified API for many hosted models
2. **Transcript chunking** — handles transcripts longer than 50k tokens
3. **GitHub PR generator** — auto-creates PRs for doc updates
4. **VS Code extension** — right-click transcript → generate docs

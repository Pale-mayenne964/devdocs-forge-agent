# Release Notes — v0.1.1

**Released:** 2026-05-14

v0.1.1 is a polish release focused on CI reliability, documentation quality, and developer experience. No breaking changes from v0.1.0.

## What Changed

### CI and Build

- Upgraded CI matrix to Node.js 22 and 24 (dropped 18, 20 — still supported at runtime per `engines.node`)
- Added macOS runner to CI matrix alongside Ubuntu
- Fixed `.env.example` typo (`ANTRHOPIC_API_KEY` → `ANTHROPIC_API_KEY`)
- README badge alignment fixed

### Documentation

- Added `docs/output-contract.md` — formal specification of the output directory structure and file guarantees
- Added `docs/safety-and-attribution.md` — safety design decisions and user responsibilities
- Added `docs/provider-system.md` — provider interface, registry, and how to add a new provider
- Added `docs/recruiter-review-guide.md` — guide for evaluating the project as a portfolio piece
- Added Mermaid architecture diagram to `docs/ARCHITECTURE.md`
- Improved `docs/GOOD_FIRST_ISSUES.md` with detailed acceptance criteria for each issue

### Example Packs

Added three complete example packs under `examples/`:

| Pack | Demonstrates |
|---|---|
| `examples/angular-signals/` | Tutorial transcript → Docusaurus page |
| `examples/api-demo/` | Demo transcript → step-by-step guide |
| `examples/product-walkthrough/` | Product demo → onboarding doc |

Each pack contains: `input.md`, `output-sample.md`, `metadata-sample.json`, `review-checklist.md`.

### README

- Added "What This Proves" section for recruiters and hiring managers
- Improved "Why Developers Star This" section
- Added Mermaid architecture diagram
- Tightened hero section copy

## No Breaking Changes

- All CLI commands, flags, and output structure are identical to v0.1.0
- `config/devdocs-forge-agent.yml` schema is unchanged
- `.env` variable names are unchanged
- Provider interface is unchanged

## Known Limitations

- `npm run preview` (browser preview of generated docs) is in review — not included in v0.1.1
- Ollama provider not yet implemented — tracked in [issue #1](https://github.com/AnkitParekh007/devdocs-forge-agent/issues/1)
- OpenRouter provider not yet implemented — tracked in [issue #2](https://github.com/AnkitParekh007/devdocs-forge-agent/issues/2)
- Transcript chunking for long videos not yet implemented — tracked in roadmap

## Upgrading from v0.1.0

```bash
git pull
npm install
npm run doctor
```

No config migration needed.

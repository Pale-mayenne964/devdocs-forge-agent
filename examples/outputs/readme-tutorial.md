Minimal, practical reference for using Angular Signals in your components.

## Quick Start

```bash
npm install @angular/core@17
```

Signals are included in `@angular/core` — no additional packages needed.

## Usage

```typescript
import { signal, computed } from '@angular/core';

// Create
const count = signal(0);

// Read
count(); // 0

// Write
count.set(5);
count.update(prev => prev + 1);

// Derive
const doubleCount = computed(() => count() * 2);
```

## In a Component

```typescript
import { Component, signal, computed } from '@angular/core';

@Component({
  selector: 'app-counter',
  template: `
    <p>Count: {{ count() }}</p>
    <p>Double: {{ doubleCount() }}</p>
    <button (click)="increment()">+1</button>
  `,
})
export class CounterComponent {
  count = signal(0);
  doubleCount = computed(() => this.count() * 2);

  increment() {
    this.count.update(v => v + 1);
  }
}
```

## API Reference

| API | Description |
|-----|-------------|
| `signal(value)` | Create a writable Signal |
| `signal.set(value)` | Replace the current value |
| `signal.update(fn)` | Update based on previous value |
| `signal()` | Read the current value |
| `computed(() => expr)` | Create a read-only derived Signal |
| `toSignal(obs$)` | Convert an Observable to a Signal |
